# Practice Django!

> Django를 사용해서 간단한 웹 서비스를 만드는 과정을 반복 연습하기 위한 용도의 저장소

## install

```shell
$ python -m venv venv
$ .\venv\Scripts\activate
$ (venv) pip install -r requirements.txt
```

## run

```
$ python .\manage.py migrate
$ python .\manage.py createsuperuser
$ python .\manage.py runserver
```
